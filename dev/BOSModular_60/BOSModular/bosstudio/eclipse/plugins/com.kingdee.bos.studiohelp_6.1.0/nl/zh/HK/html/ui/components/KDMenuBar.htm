功能列表

 請注意當你看到這句話時，菜單欄還沒實現向IE瀏覽器那樣的功能，在6月30號之前完成。

 

菜單欄（KDMenuBar）的佈局方式IE 瀏覽器一樣。

 

菜單欄(com.kingdee.bos.ctrl.swing.KDMenuBar) 添加的菜單默認情況下是靠左佈局的，但現在提供擴展接口<code>addRightComponent(JComponent menu,int rightInterval)</code> 可將一個Swing控件插入到靠右端的位置。應用場景如:在菜單欄的右邊插入一個幫助按鈕（參見EAS3.0的主界面）(在4月30號之前提供)

菜單欄可以添加一個Logo , 參見接口<code>setLogo(Icon)</code> (在4月30號之前提供)

菜單欄可在同一個方位區上移動，移動的方式為：與工具欄進行交換行顯示或者交換列顯示。

菜單欄上的菜單(KDMenu)，全部顯示或隱藏顯示；

當窗口大小改變時：()

1，  菜單欄在北區、南區時，窗口的寬度不足以水平的顯示菜單欄上的所有菜單，則菜單欄的高度改變，菜單在菜單欄上隱藏一部分顯示； (在4月30號之前提供)

2，  菜單欄在東區、西區時，窗口的高度不足以垂直的顯示菜單欄上的所有菜單，則菜單欄的寬度改變，菜單在菜單欄上隱藏一部分菜單顯示；  (在4月30號之前提供)

菜單欄以浮動窗口方式顯示 。(在4月30號之前提供)

菜單欄可拖拉到四個區進行佈局

菜單的快捷鍵操作與word和永中office一樣

1：Alt鍵 激活菜單欄上的第一個菜單（比如word中的「文件(F)」）

2：在激活第一個菜單後，可用「左右箭頭」在菜單欄上導航菜單。

3：在激活第一個菜單後，可由「相關鍵「直接選中某個菜單，並彈出該菜單。比如按下「E」，會把「編輯(E)」菜單擊中，並彈出「編輯(E)」菜單。

4：在選中一個菜單後，可用「上下箭頭」進行導航菜單項（MenuItem）

 

5：Alt＋「某鍵」，選中某菜單，並彈出某菜單。比如同時按下「Alt+F」鍵，擊中「文件（F）」菜單，並彈出該菜單。

默認情況下菜單是從左至右佈局，提供接口把指定的菜單從右至左佈局。

 

菜單的行為方式，菜單項的行為方式，與標準JMenu，JMenuItem完全兼容。比如

1：菜單的打開。

2：根據index從MenuBar上得到指定的Menu。

3，根據index從Jmenu上得到JmenuItem。等等。

 

KDMenuBar從javax.swing.JMenuBar繼承。

KingdeeMenuBarUI從BasicMenuBarUI繼承。

僅構造新的佈局管理器，來完成KDMenuBar的要求。

 

使用指南

 

//1 構造菜單欄

    private KDMenuBar getKDMenuBar() {

       if (menuBar == null) {

           menuBar = new KDMenuBar();

 

           //1 File

           KDMenu menuFile = new KDMenu("File");

           KDMenuItem itemNew = new KDMenuItem("New");

           KDMenuItem itemSave = new KDMenuItem("Save");

           KDMenuItem itemSaveAs = new KDMenuItem("saveAs");

 

           menuFile.add(itemNew);

           menuFile.add(itemSave);

           menuFile.add(itemSaveAs);

 

           //2

           KDMenu menuEdit = new KDMenu("Edit");

           KDMenuItem itemUndo = new KDMenuItem("Undo");

           KDMenuItem itemRedo = new KDMenuItem("Redo");

           KDMenuItem itemCut = new KDMenuItem("Cut");

           menuEdit.add(itemUndo);

           menuEdit.add(itemRedo);

           menuEdit.add(itemCut);

 

 

           //6

           menuBar.add(menuFile);

           menuBar.add(menuEdit);

    。。。。

 

       }

       return menuBar;

   }

 

// 給窗體設置菜單欄

KDFrame mainframe = new KDFrame();

mainFrame.setJMenuBar(getKDMenuBar());
