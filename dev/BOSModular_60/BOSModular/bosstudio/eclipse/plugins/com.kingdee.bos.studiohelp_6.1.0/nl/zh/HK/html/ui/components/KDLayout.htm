功能列表

   通過為每一個控件添加一個停靠約束（Anchor），把設計時的絕對定位佈局轉化為可縮放的相對佈局。

 

使用指南

  KDLayout是專門為UIDesigner設計佈局管理器，通過對設計面板上的每一控件添加一個anchor屬性，KDLayout就能將UIDesigner上絕對定位的佈局變成可縮放的佈局。如果在UIDesigner外單獨使用KDLayout，你需要填寫更多的屬性。

 

KDLayout佈局屬性

       KDLayout的內部類Constraints定義了在使用KDLayout時容器上每個控件必須填寫的佈局屬性。

 

OriginalBound

       表示的是控件在UIDesigner上的原始大小

Anchor

       表示的是控件的錨定方式。

       錨定是指控件上的一邊與父容器的同方向的一邊或是中心點之間的固定關係。當控件的錨定後，控件位置相對父容器固定。當父容器大小改變後，控件與父容器之間的固定關係不變，控件的位置大小由父容器所決定。

在KDLayout中定義了五個錨定點、兩種錨定方式，共九個錨定方式可選擇。不同錨定的方式用不同的常量表示，而且可以通過邏輯運算符「或」進行搭配使用。

    ANCHOR_TOP ：

控件的上邊與父容器的上邊距離固定，等同於設計時距離

    ANCHOR_BOTTOM ：

控件的下邊與父容器的下邊距離固定，等同於設計時距離

    ANCHOR_LEFT ：

控件的左邊與父容器的左邊距離固定，等同於設計時距離

ANCHOR_RIGHT ：

控件的右邊與父容器的右邊距離固定，等同於設計時距離

ANCHOR_CENTRE ：

控件的中心點與父容器的中心點水平距離固定，等同於設計時距離

    ANCHOR_TOP_SCALE :

控件的上邊與父容器的上邊距離固定，等同於設計時距離乘【父容器高度與設計時高度的縮放比】。

    ANCHOR_BOTTOM_SCALE :

控件的下邊與父容器的下邊距離固定，等同於設計時距離乘【父容器高度與設計時高度的縮放比】。

    ANCHOR_LEFT_SCALE :

控件的左邊與父容器的左邊距離固定，等同於設計時距離乘【父容器寬度與設計時寬度的縮放比】。

    ANCHOR_RIGHT_SCALE :

控件的右邊與父容器的右邊距離固定，等同於設計時距離乘【父容器寬度與設計時寬度的縮放比】。

 

  除了每個控件的佈局信息，控件的佈局容器也需要填寫一個ClientProperty：「OriginalBounds」用於記錄UIDesigner設計時父容器（設計面板）的大小。

      

API說明

參見產生的通用控件JavaDoc文檔。

 

示例

       三個鍵鈕，一個左對齊、一個右對齊、一個居中

       cons = new KDLayout.Constraints();

    cons.originalBounds = new Rectangle(19, 27, 71, 25);

    cons.anchor = KDLayout.Constraints.ANCHOR_LEFT;

    b11.putClientProperty("KDLayoutConstraints", cons);

   

 

    cons = new KDLayout.Constraints();

    cons.originalBounds = new Rectangle(162, 27, 71, 25);

    cons.anchor = KDLayout.Constraints.ANCHOR_CENTRE_SCALE;  

    b12.putClientProperty("KDLayoutConstraints", cons);

   

 

    cons = new KDLayout.Constraints();

    cons.originalBounds = new Rectangle(314, 26, 71, 25);

    cons.anchor = KDLayout.Constraints.ANCHOR_RIGHT;  

b13.putClientProperty("KDLayoutConstraints", cons);

 

 

    getContentPane().add(b13, null);

    getContentPane().add(b12, null);

    getContentPane().add(b11, null);

           

// 填寫父容器設計時的大小

((JComponent)this.getContentPane()).putClientProperty("OriginalBounds", new Rectangle(0, 0, 397, 297));

getContentPane().setLayout(new KDLayout());

 

// 設置容器的最佳大小，當容器大小小於最佳大小時出現滾動條

((JComponent)getContentPane()).setPreferredSize(new Dimension(397, 297));

setContentPane(new JScrollPane(getContentPane())); 

 

 

雖然在上面的例子中控件的佈局約束是通過ClientProperty的方式添加的；但是KDLayout也支持按傳統的通過JComponent.add(Component,Object)的方式添加控件約束。
