使用指南

一、圖表的類型

     1、圖表類別,詳見ChartCategory類，目前支持的類別有：

        ·PIECHART，餅圖

        ·BARCHART，柱形圖/條形圖

              ·LINECHART，折線圖

        ·SCATTERPLOT，散點圖

        ·AREACHART，面積圖

        ·GANTTCHART，甘特圖

  

2、圖表類型，具體見ChartType類，目前支持的類型有：

·CT_PIE, 餅圖，屬於PIECHART

·CT_PIEEXPLODED， 分離型餅圖，屬於PIECHART

           ·CT_PIE3D, 三維餅圖，屬於PIECHART

           ·CT_COLUMNCLUSTERED, 簇狀柱狀圖，屬於BARCHART

·CT_COLUMNSTACKED，堆積柱形圖，屬於BARCHART

·CT_COLUMNCLUSTERED3D，三維簇狀柱形圖，屬於BARCHART

·CT_COLUMNSTACKED3D，三維堆積柱型圖，屬於BARCHART

·CT_BARCLUSTERED，簇狀條形圖，屬於BARCHART

·CT_BARSTACKED，堆積條形圖，屬於BARCHART

·CT_BARCLUSTERED3D，三維簇狀條形圖，屬於BARCHART

·CT_BARSTACKED3D，三維堆積條型圖，屬於BARCHART

           ·CT_LINE, 折線圖，屬於LINECHART

           ·CT_LINEMARKERS, 數據點折線圖，屬於LINECHART

           ·CT_LINESTACKED, 堆積折線圖，屬於LINECHART

           ·CT_LINEMARKERSSTACKED, 數據點堆積折線圖，屬於LINECHART

           ·CT_XYSCATTER, 散點圖，屬於SCATTERPLOT

           ·CT_XYSCATTERLINES, 折線散點圖，屬於SCATTERPLOT

·CT_XYSCATTERLINESNOMARKERS, 無數據點折線散點圖

  ，屬於SCATTERPLOT

·CT_AREA，面積圖，屬於AREACHART

·CT_AREASTACKED，堆積面積圖，屬於AREACHART

·CT_GANTT, 甘特圖，屬於GANTTCHART

 

      註：通過使用setChartType(ChartType chartType)方法來設置和改變圖表類型

 

    二、圖表的數據

    ·圖表數據類型共有三種：

    1、PieChartData, 餅圖專用數據存儲結構

       ·groupKeys, 組別名數組

·values，圖表數據，一維數組。

         groupKeys和values維同大小一維數組，一個組別名對應一個數據

    2、CommonChartData, 通用數據存儲結構

·seriesKeys, 序列名數組

·groupKeys, 每個序列中的組名

·values, 圖表數據，一行為一個序列，一列為一組。

    3、GanttChartData, 甘特圖專用數據存儲結構

·seriesKeys, 序列名數組

        ·groupKeys, 每個序列中的組名

·dates, 圖表日期數組數據，此數組為三維數組，

一行為一個序列，一列為一組，每個單元為一個二維數組，

包含了開始時間和結束時間 。

   

4、ChartDataUtil，用於構建圖表數據的實用工具

       ·createPieChartData(String[] groupKeys, double[] values);

    ·createCommonChartData(String[] seriesKeys, String[] groupKeys, double[][] values);

    ·createGanttChartData(String serieskeys, String[] groupKeys, Date[][][] dates);

 

註：通過使用下列方法對圖表數據進行操作

    addChartData(IChartData chartData);

    removeChartData(int index);

    getChartData(int index);

   

圖表類型為散點圖時，values數組的第0行是x值，第1行之後是各個序列對應的y值。比如下面數組：

  {{1,   2,  3,  4,  5,  6,  7},

   {12, 23, 52, 13,  4,  9, 17},

   {22, 33, 51, 17, 14, 19, 31} }

該數組表示兩個序列，序列的x值是第0行，y值是後兩行，

所組成的散點坐標序列為：

序列1：(1,12),(2,23),(3,52),(4,13),(5, 4),(6, 9),(7,17)

序列2：(1,22),(2,33),(3,51),(4,17),(5,14),(6,19),(7,31)

 

 

三、圖表

    1、構建KDChart對象，

       KDChart chart = new KDChart();

       Or

       KDChart chart = new KDChart(ChartType.CT_PIE);

 

2、設置圖表類型（如果構建時候沒有指定圖表類型，需要設置圖表類型）。

   chart.setChartType(ChartType.CT_PIE3D);

 

3、設置圖表標題

   ChartTitle chartTitle = chart.getChartTitle();

   chartTitle.setText(「text title」);

   chartTitle.setFonr(new KDFont(「細明體」, KDFont.Plaint, 12);

   chartTitle.setPaint(Color.blue);

   chartTitle.setBackgroundPaint(Color.green);

 

4、設置坐標軸標題

chart.getAxes(0).getCategotyAxis().getTitle().setText("Category Axis Label");

               chart.getAxes(0).getValueAxis().getTitle().setText("Value Axis Label");

 

5、圖表數據

   PieChartData pieChartData = ChartDataUtil.createPieChartData(

                            String[] groupKeys, double[] values);

   Chart.addChartData(pieChartData);

 

創建一個圖表典型例程如下：

chart = new KDChart();

chart.setChartType(ChartType.CT_PIE);

chart.getChartTitle().setText("My Chart Title");

chart.getAxis(AxisType.CATEGORY, chart.getAxes(0).getCategotyAxis().getTitle().setText("Category Axis Label");

chart.getAxes(0).getValueAxis().getTitle().setText("Value Axis Label");

DemoDataTable data = new DemoDataTable();

PieChartData pieChartData = ChartDataUtil.createPieChartData(

                         String[] groupKeys, double[] values);

Chart.addChartDate(pieChartData);
