使用指南

日期選擇框能進行日期和時間的編輯，默認情況下只能進行日期選擇「××××年××月××日」，可通過調用用函數setTimeEnabled(boolean)來設置是否也有時間編輯。對日期進行編輯時，可手工直接在編輯框裡進行改動，也可點擊小按鈕彈出一個日期選擇表框進行選擇。

日期選擇表框彈出後，是一個時尚的設計。年的範圍可從1900至2099年，月從一月至12月。並且有「是否返回空值」的可選設置。

該控件接受的數據類型必須是java.util.Date及其子類java.sql.Date,java.sql.Timestamp.

 

1．常用用法

KDDatePicker datePicker = new KDDatePicker();

 

 

2．設置時間是否顯示

   datePicker.setTimeEnabled(true); // 時間「時：分：秒」可顯示

   datePicker.setTimeEnabled(false); // 時間「時：分：秒」不可顯示
