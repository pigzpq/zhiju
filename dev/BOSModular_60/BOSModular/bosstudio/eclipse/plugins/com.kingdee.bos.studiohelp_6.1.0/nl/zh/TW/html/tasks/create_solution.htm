創建一個解決方案

針對BOS而言，解決方案是元數據的最大載體，所有的元數據都必須位於一個解決方案之內，譬如，EAS就是一個解決方案；從Eclipse的角度來說，解決方案就是Eclipse的一個Project。因此，解決方案本身在BOS Studio中就具備Project的一些性質，譬如導入、導出、刷新等。

在BOS Studio中，解決方案能夠新建、打開、關閉、刪除。這些操作均通過「BOS資源導航視圖」完成。

如何創建解決方案

1、打開「包分類」視圖，在右鍵菜單中選擇「新建」->「解決方案」，彈出「解決方案創建嚮導」，如圖一：

圖一

2、輸入解決方案的「名稱」、「別名」和「描述」，確定解決方案的放置位置，點擊「下一步」，如圖二：

圖二

3、在「目錄(D)」欄中輸入元數據放置的目錄，點擊「完成」，即可建立一個解決方案。在「包分類」視圖中可看見結果，如圖三：
