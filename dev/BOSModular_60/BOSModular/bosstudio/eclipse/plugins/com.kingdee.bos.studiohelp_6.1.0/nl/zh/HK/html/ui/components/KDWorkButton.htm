支持圖標（1）+文本（1字串帶html標籤）方式

鼠標左鍵按下顯示下沉效果

　

用戶操作

控件支持tab鍵的順序切換

控件焦點

切換按鈕顯示風格

操作區按鈕支持右鍵方式選擇按鈕風格模式：純文字，純圖標，文字＋圖標

Enter鍵按下，觸發點擊

當前具有焦點（即 激活狀態）的按鈕，必須支持鍵盤 Enter 鍵操作（即按 Enter 鍵後自動執行相應操作）；

Space鍵按下同鼠標左鍵按下

　

Space鍵放開同鼠標左鍵放開

使用指南

       操作區按鈕是一個特殊的形式按鈕，並支持顯示風格的設置：

       屬性的使用：

·Style屬性：顯示風格，如下：

1）STYLE_ICON 顯示圖標

2）STYLE_TEXT 顯示文本

STYLE_ICON|STYLE_TEXT 顯示圖標和文本。

另外：按鈕的大小約束見人機工程部文檔。
