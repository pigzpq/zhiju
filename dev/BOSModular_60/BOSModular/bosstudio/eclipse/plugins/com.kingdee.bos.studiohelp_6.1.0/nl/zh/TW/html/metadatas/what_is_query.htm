查詢簡介

什麼是查詢？這裡將向你作簡單介紹。

當建好實體對像、關係對像和表格對像後，根據需要就可以創建一個Query對像提供對數據的展示。

BOS中的Query對象是實體對象的查詢視圖，可以在數據訪問過程中只獲取視圖中指定屬性的數據，並按照過濾、排序條件進行數據的過濾和排序。

Query對象的數據組織形式是二維模型的，Query對象可以分為連接查詢對像和聯合查詢對象。在一個連接查詢對像中(Join Query)可以包含多個子對像:實體(Entity)或者其他的查詢對像(Sub Query稱為子查詢)，但是在一個連接查詢對像中有且僅有一個主對象。在一個聯合查詢對像(Union Query)中，包含的子對象都必須是一個(Sub Query 稱為子查詢)，並且聯合查詢對像默認第一個選中的子對像為主對象。
