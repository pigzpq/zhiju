如何從一個編輯器(Editor)中快速定位其它的對象

如何從一個編輯器(Editor)中快速定位其它的對象？比如，如何從實體中看到對應的表對像？

我們在一個Editor中，經常需要查看與該Editor相關的其它Editor。譬如在實體MyEntity定義界面中，想快速的定位到MyEntity對應的表對像T_MyTable，可以如下操作：在對應表對象的TextEdit框中，鼠標懸停，會發現有提示：hold CTRL and click，就是說：如果用鼠標點一下TextEdit框（先讓該TextEdit框得到焦點），然後按住Ctrl鍵，TextEdit框中的字會變成超鏈形式（字體變藍並加下劃線），然後點擊鼠標，即可打開該TextEdit框所描述的文件。
