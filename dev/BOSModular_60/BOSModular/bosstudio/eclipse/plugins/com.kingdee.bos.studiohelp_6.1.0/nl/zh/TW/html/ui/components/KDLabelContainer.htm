控件名稱

組合標籤控件

類名

com.kingdee.bos.ctrl.swing.KDLabelContainer

父類

javax.swing.JComponent

父接口

com.kingdee.bos.ctrl.swing.IKDLabelContainer

 

相關的類

 

 

 

功能列表

該控件提供一個容器，對一個標籤和一個控件進行綁定，並佈局。

佈局思想如下：組合控件默認大小為標籤大小與輸入控件大小之和。若用戶設置組合控件大小，則優先佈局標籤，剩餘空間分配給輸入控件。

提供下列五種佈局方式（標籤相對於輸入控件）：西（默認值）、東、北、西北、東北。可通過setBoundLabelAlignment(int)設置。

可設置標籤的大小，setLabelLength在不同的佈局方式下有不同含義：當佈局方式為水平向（即東、西），該參數表示標籤的寬度；當佈局方式為垂直向，該參數表示標籤的高度。

IKDLabelContainer定義如下接口:

//設置被綁定的輸入控件

public void setBoundEditor(JComponent editor);

public JComponent getBoundEditor();

 

//以下8個是設置label的接口：

//設置被綁定的標籤(默認自帶標籤，可不用設)

public void setBoundLabel(JLabel label);

public JLabel getBoundLabel();

 

//設置標籤的佈局方式

//標籤相對於editor有五種方位，參數取值分別如下：

//SwingConstants.WEST(默認)

//SwingConstants.EAST

//SwingConstants.NORTH

//SwingConstants.NORTH_WEST

//SwingConstants.NORTH_EAST

public void setBoundLabelAlignment(int alignment);

public int getBoundLabelAlignment();

 

//設置標籤的長度（或高度，視labelAligment而定）

public void setBoundLabelLength(int len);

public int getBoundLabelLength();

 

//設置標籤的文本

public void setBoundLabelText(String labelText);

public String getBoundLabelText();

 

對於KDTextField、KDTextArea、KDSpinner、KDSlider、KDPassword、KDList、KDComboBox等控件，實現了IBoundLabelControl接口，那麼當它們被放到KDLabelContainer中之後，可以從這些輸入控件直接設置標籤。即不需要知道組合標籤控件，如下例中那樣：text. setLabelLength(30)。

 

使用指南

 

例子：   

       KDTextField text =  new KDTextField();                //創建一個輸入控件

KDLabelContainer c = new KDLabelContainer ();          //創建組合標籤控件

c. setBoundEditor (text);                              //將輸入控件添加進來

 

c. setBoundLabelText (「請輸入XXX：」);                   //設置標籤文字

c. setBoundLabelLength (30);                              //設置標籤高度

c. setBoundLabelAlignment (SwingConstants.NORTH_WEST);  //設置標籤位置

//由於KDTextField也實現了設置標籤的8個接口，所以也可以這麼用：

text. setBoundLabelText (「請輸入XXX：」);                   //設置標籤文字

text. setBoundLabelLength (30);                              //設置標籤高度

text. setBoundLabelAlignment (SwingConstants.NORTH_WEST);  //設置標籤位置
