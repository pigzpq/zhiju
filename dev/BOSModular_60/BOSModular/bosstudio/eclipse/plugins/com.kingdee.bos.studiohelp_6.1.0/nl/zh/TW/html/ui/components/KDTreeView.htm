★       KDTreeView與KDTree

KDTreeView 實現 了對KDTree的包裝。將KDTree置於KDScrollPane中，並加上標題欄和操作按扭。

例：

KDTreeView treeView = new KDTreeView(tree);

 

例：

KDTreeView treeView = new KDTreeView();

treeView.setTree(tree);

 

取得KDTree：

treeView.getTree()
