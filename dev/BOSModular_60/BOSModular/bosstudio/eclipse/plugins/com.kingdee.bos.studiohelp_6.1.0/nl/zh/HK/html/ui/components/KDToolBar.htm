功能列表

   基本功能請參見 javax.swing.JToolBar 類說明

 

工具欄和word、永中office一樣。

工具欄KDToolBar繼承與javax.swing.JToolBar，完全與JToolBar兼容。僅構造新的佈局算法滿足工具欄需求。

新加public 方法 addWorkButton(KDWorkButton) ，添加一個工作區按鈕(KDWorkButton 工作區按鈕請參見工作區按鈕文檔說明)。

 

重載public 方法 public void add(Action) 方法，其實返回的是一個工作區按鈕（KDWorkButton）

 

提供擴展接口<code>addRightComponent(JComponent menu,int rightInterval)</code> 可將一個Swing控件插入到靠右端的位置。該功能在4月30號提供

 

在同一個區類移動，移動的方式為：與菜單欄換行顯示。

工具欄可追尾添加。

1：當在北區和南區時，一行上可佈局一個至多個工具欄。

2：當在東區和西區時，一列上可佈局一個至多個工具欄。該功能在4月30號提供

 

工具欄可在不同的區佈局，默認的情況下是在北區水平的顯示。

工具欄能以浮動窗口顯示。該功能在4月30號提供

 

工具欄的最末端出現一個小按鈕（這一點參見永中office）。

工具欄的寬度或高度不能全部顯示工具欄上的控件項時，將

1：水平時 將在工具欄的最右端出現一個小按鈕，點擊小按鈕出現一個工具欄小面板（ToolBarListPane），小面板上容納不能在工具欄上顯示的控件項。

2：垂直時 將在工具欄的最下端出現一個小按鈕，點擊小按鈕出現一個工具欄小面板（ToolBarListPane），小面板上容納不能在工具欄上顯示的控件項。該功能在4月30號提供

 

當工具欄的高寬改變時：

1，  水平佈局時，若寬度從小的不能顯示其上所有控件到能夠顯示其上所有控件項時，小面板上的控件項自動添加到工具欄上。

2，  垂直佈局時，若高度從小的不能顯示其上所有控件到能夠顯示其上所有控件項時，小面板上的控件項自動添加到工具欄上。

 

 

 

使用指南

  基本用法見javax.swing.JToolBar用法。

// 添加一個工作區按鈕

KDWorkButton workButton ＝ new KDWorkButton(「new」);

kdToolBar.add(workButton);

 

//添加一個Action

MyAction myAction ＝ 。。。

KDWorkButton createBu = kdToolBar.add(myAction);

 

// 往工具欄的右側添加一個距離右側5個像數的按鈕

KDWorkButton button = new KDWorkButton();

kdToolBar. addRightComponent(button,5);
