★     KDTree的模與節點

KDTree從JTree繼承，它的模式及結構也是與JTree保持一致，而JTree使用的模DefaultTreeModel和節點DefaultMultableTreeNode不能滿足KDTree的需求，於是擴展了KingdeeTreeModel和DefaultKingTreeNode。

 

       JTree   － DefaultTreeModel  － DefaultMutableTreeNode

       KDTree － KingdeeTreeModel － DefaultKingdeeTreeNode

 

使用KDTree，必須使用KingdeeTreeModel和DefaultKingdeeTreeNode。

 

★     樹的創建

常用的構造函數：

       KDTree()

       KDTree(ITreeNode root)

       KDTree(TreeModel newModel)

例：

DefaultKingdeeTreeNode root = new DefaultKingdeeTreeNode("RootNode");

KDTree tree = new KDTree(root);

例：

DefaultKingdeeTreeNode root = new DefaultKingdeeTreeNode("RootNode");

KDTree tree = new KDTree(new KingdeeTreeModel(root));

例：

DefaultKingdeeTreeNode root = new DefaultKingdeeTreeNode("RootNode");

KDTree tree = new KDTree();

tree.setModel(new KingdeeTreeModel(root));

FAQ：（問題徵集中，歡迎提出使用中的任何問題。）

1、 KDTree中的一些節點，我想讓它成為文件夾，可是當它沒有子節點時，它會顯示葉子節點的圖標。是否有辦法指定它是文件夾圖標？

答：可以。我們首先想的辦法可能是用DefaultKingdeeTreeNode的setCustomIcon方法來設節點的自定義圖標。但是這個圖標不能根據文件夾是否打開的狀態而變化。實際上JTree已經提供了解決方案。

首先，treeModel中有一個asksAllowsChildren的屬性，默認是false，我們要把它設為true：

((KingdeeTreeModel)tree.getModel()).setAsksAllowsChildren(true);

然後，對於希望成為文件夾的節點node1:

node1.setAllowsChildren(true); //其實默認已經就是true

這樣，不管node1有沒有子節點，都會以文件夾圖標出現。且雙擊它使其打開/關閉時，圖標也會變化。

 

2、 當只有一個根節點並且根節點被隱藏時，不能往根節點添加子節點。而根節點沒有隱藏，或者根節點隱藏了，但它至少有一個子節點時，添加子節點的操作一切都正常。是什麼原因？

答：並不是「不能添加子節點」，只是此時根節點沒有展開，看不到結果而已。只要把根節點展開就行了：

tree.expandAllNodes(true, (DefaultKingdeeTreeNode)tree.getModel().getRoot());

 

3、 刪除節點時，界面上為什麼不刷新？而且我用了tree.validate(); tree.repaint();也無效。

答：這種情況很可能是這樣使用造成的：

parentNode.remove(childNode);

因為從節點的角度去移除節點，樹的模並不知道，所以會有「刪不了」的情況。要改成這樣：

tree.removeNodeFromParent(childNode);

順便再說一下，KDTree提供了增刪節點的幾個方法，都是從模去操作的。這幾個方法如下：

//刪除節點:

public void removeNodeFromParent(MutableTreeNode node)

//刪除節點的所有子節點:

public void removeAllChildrenFromParent(MutableTreeNode parent)

//插入節點:

public void insertNodeInto(MutableTreeNode newChild,MutableTreeNode parent,int index)

//在父節點末端插入節點:

public void addNodeInto(MutableTreeNode newChild,MutableTreeNode parent)
