使用指南

       自定義表盤控件(KDCustomDial)繼承自基類(BasicDial)，支持多段標刻。基類的使用可以參見示例(TestBasic)。下面主要介紹KDCustomDial的使用。

       構造一個自定義表盤，可以使用KDCustomDial的構造函數，KDCustonDial類提供了兩個構造函數：

· public KDCustomDial(CustomDialRange range,String title){}

   參數：

range ----- 表盤的總範圍

title  ----- 表盤的標題

 

例子：

CustomDialRange range = new CustomDialRange(「總範圍」,Color.BLACK,0,180);

KDCustomDial dial = new KDCustomDial(range,」KDCustomDial演示」);

 

· public KDCUstomDial(CustomDialRrange[] ranges,String title){}

   參數:

ranges ----- 表盤的範圍數組，其中第一個元素（ranges[0]）為表盤的

              總範圍。

title ----- 表盤的標題。

      

       例子：CustomDialRange[] ranges = {

              New CustomDialRange(「總範圍」,Color.BLACK,0,180),

              New CustomDialRange(「範圍1」,Color.RED,0,30),

              New CustomDialRange(「範圍2」,Color.BLUE,30,60),

              New CustomDialRange(「範圍3」,Color.GREEN,60,90),

              New CustomDialRange(「範圍4」,Color.GRAY,90,140)

};

KDCustomDial dial = new KDCustomDial(ranges,」KDCustomDial演示」);

 

下面是自定義表盤的方法說明：

       一、對表盤的值進行設置。使用方法：

              KDCustomDial.setValue(Object value);

              value: 取Double類型。

二、對表盤外觀的定制

取得表盤中的組件對象，進行屬性設置。KPI表盤中的組件默認序號為0。

例子1：設置刻度盤不同範圍的顏色

a. 獲得刻度盤對像：CustomDialScale scl = KDCustomDial.getScale();

b. 獲取要設置顏色的範圍：CustomDialRange range = scl.getRange(i);

   i為索引，整型(int)。

c. 定制某個範圍的顏色：range.setColor(Color.BLACK);

例子2：設置刻度盤標籤顯示格式

dial.getScale().setTickLabelFormat(new DecimalFormat("#"));

默認顯示格式為：DecimalFormat("###』%』");

例子3：設置刻度盤標題

String title =」KDCustomDial演示」);

KpiTitle ttl = new KpiTitle(title);

dial.setTitle(ttl);

例子4：獲得表盤的示例圖

CustomDialLegend legend = dial.createLegend();

CustomDialLegend legend = dial.createLegend(int orientation);

Orientation 一般取值

CustomDialLegend.VERTICAL 或者

CustomDialLegend.HORIZONTAL

三、 定制多個範圍

. 如果想要對表盤範圍進行添加、刪除、清除操作，首先必須先得到表盤的刻度盤對象。如下：

              CustomDialScale scale = dial.getScale();

1． 添加一個範圍

使用CustomDialScale的addRange(CustomDialRange range)方法

如：scale.addRange(new CustomDialRange(「範圍5」,Color.GRAY,140,180);

2.        添加多個範圍

使用CustomDialScale的addRanges(CustomDialRange[] ranges)方法

3.        刪除一個範圍

CusotmDialScale.removeRange(int index);  index為範圍列表的索引

CustomDialScale.removeRange(CustomDialRange range);

4.        清空表盤的範圍

CustomDialScale.clearRanges();

 

 

示例

 

 

package ctrldemo.swing.dial;

 

import java.awt.Color;

import java.awt.Container;

import java.awt.Graphics;

import java.awt.event.ActionEvent;

import java.awt.event.ActionListener;

import java.awt.image.BufferedImage;

 

import javax.swing.JButton;

import javax.swing.JDialog;

import javax.swing.JPanel;

 

import com.kingdee.bos.ctrl.swing.KDCustomDial;

import com.kingdee.bos.ctrl.swing.dial.CustomDialLegend;

import com.kingdee.bos.ctrl.swing.dial.CustomDialRange;

import com.kingdee.bos.ctrl.swing.dial.CustomDialScale;

 

import ctrldemo.swing.backup.TestingFrame;

 

/**

 * @author 蘇志欽

 * 

 */

public class TestKDCustomDial extends TestingFrame

{

       KDCustomDial dial;

       public TestKDCustomDial()

       {

              //表盤範圍數組

              CustomDialRange[] ranges = {

              new CustomDialRange("總範圍",Color.BLACK,0, 180),

              new CustomDialRange("範圍1",Color.BLUE,0, 30),

              new CustomDialRange("範圍2",Color.RED,30, 50),

              new CustomDialRange("範圍3",Color.GREEN,50, 90),

              new CustomDialRange("範圍4",Color.WHITE,90,140)

              };

              //構造自定義表盤

              dial = new KDCustomDial(ranges,"KDCustomDial Demo");

              dial.setValue(new Double(90));

             

              //添加一個範圍

              CustomDialScale scale = dial.getScale();

              scale.addRange(new CustomDialRange("範圍5",Color.CYAN,140,180));

              cp.add(dial);

             

              //創建示例圖

              CustomDialLegend legend = dial.createLegend();

              cp.add(legend);

             

              JButton btnimg = new JButton("保存圖片");

              cp.add(btnimg);

              btnimg.addActionListener(new ActionListener()

              {

                     BufferedImage img = null;

 

                     public void actionPerformed(ActionEvent e)

                     {

                            JDialog w = new JDialog();

                            Container cp = w.getContentPane();

                            img = dial.createBufferedImage();

                            JPanel p = new JPanel()

                            {

                                   public void paint(Graphics g)

                                   {

                                          g.setColor(Color.WHITE);

                                          g.fillRect(0, 0, getWidth(), getHeight());

                                          g.drawImage(img, 0, 0, this);

                                   }

                            };

                            p.setPreferredSize(dial.getSize());

                            cp.add(p);

                            w.show();

                            w.pack();

                     }

              });         

       }

 

       public static void main(String[] args) throws Exception

       {

              TestKDCustomDial test = new TestKDCustomDial();

              test.show();

       }

}
