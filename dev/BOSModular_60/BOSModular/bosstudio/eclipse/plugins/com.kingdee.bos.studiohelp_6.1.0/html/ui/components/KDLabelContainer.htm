功能列表

该控件提供一个容器，对一个标签和一个控件进行绑定，并布局。

布局思想如下：组合控件默认大小为标签大小与输入控件大小之和。若用户设置组合控件大小，则优先布局标签，剩余空间分配给输入控件。

提供下列五种布局方式（标签相对于输入控件）：西（默认值）、东、北、西北、东北。可通过setBoundLabelAlignment(int)设置。

可设置标签的大小，setLabelLength在不同的布局方式下有不同含义：当布局方式为水平向（即东、西），该参数表示标签的宽度；当布局方式为垂直向，该参数表示标签的高度。

IKDLabelContainer定义如下接口:

//设置被绑定的输入控件

public void setBoundEditor(JComponent editor);

public JComponent getBoundEditor();

 

//以下8个是设置label的接口：

//设置被绑定的标签(默认自带标签，可不用设)

public void setBoundLabel(JLabel label);

public JLabel getBoundLabel();

 

//设置标签的布局方式

//标签相对于editor有五种方位，参数取值分别如下：

//SwingConstants.WEST(默认)

//SwingConstants.EAST

//SwingConstants.NORTH

//SwingConstants.NORTH_WEST

//SwingConstants.NORTH_EAST

public void setBoundLabelAlignment(int alignment);

public int getBoundLabelAlignment();

 

//设置标签的长度（或高度，视labelAligment而定）

public void setBoundLabelLength(int len);

public int getBoundLabelLength();

 

//设置标签的文本

public void setBoundLabelText(String labelText);

public String getBoundLabelText();

 

对于KDTextField、KDTextArea、KDSpinner、KDSlider、KDPassword、KDList、KDComboBox等控件，实现了IBoundLabelControl接口，那么当它们被放到KDLabelContainer中之后，可以从这些输入控件直接设置标签。即不需要知道组合标签控件，如下例中那样：text. setLabelLength(30)。

 

使用指南

 

例子：   

       KDTextField text =  new KDTextField();                //创建一个输入控件

KDLabelContainer c = new KDLabelContainer ();          //创建组合标签控件

c. setBoundEditor (text);                              //将输入控件添加进来

 

c. setBoundLabelText (“请输入XXX：”);                   //设置标签文字

c. setBoundLabelLength (30);                              //设置标签高度

c. setBoundLabelAlignment (SwingConstants.NORTH_WEST);  //设置标签位置

//由于KDTextField也实现了设置标签的8个接口，所以也可以这么用：

text. setBoundLabelText (“请输入XXX：”);                   //设置标签文字

text. setBoundLabelLength (30);                              //设置标签高度

text. setBoundLabelAlignment (SwingConstants.NORTH_WEST);  //设置标签位置
