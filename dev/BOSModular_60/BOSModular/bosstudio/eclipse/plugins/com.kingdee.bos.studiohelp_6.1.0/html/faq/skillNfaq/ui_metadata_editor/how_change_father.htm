如何将以前定义成从CoreUIObject继承的UI ，改成从EditUI或其它UI继承呢

假设现有 BTPMakeBillFromUI.ui文件，是从CoreUIObject继承的，现要改成从EditUI继承，具体做法如下：

A）、将BTPMakeBillFromUI.ui文件以UTF-8格式用文本编辑器打开；

B）、在<title>与<components>节点之间增加以下节点，其中package所对应的value是指父UI的包路径全称，name所对应的value是指父UI名称：
<parent>
    <key name="package" value="com.kingdee.eas.framework.client" />
    <key name="name" value="EditUI" / >
</parent>

C）、在所有<component>节点中，删除<component>节点下<name>节点值为当前UI名称、以及当前UI名称+"_toobar"、当前UI名称+"_menubar"的三个<component>点。在本例中，即删除以下三个节点：
<component>
    <name>BTPMakeBillFromUI</name>
    <classType>com.kingdee.bos.ctrl.swing.KDPanel</classType>
    <isContainer>true</isContainer>
    <constraints>10,10,300,150</constraints>
    <tabIndex>0</tabIndex>
    <herited>false</herited>
    <layoutInfo>
        <name>kDLayout1</name>
        <type>KDLayout</type>
        <value />
    </layoutInfo>
</component>
<component>
    <name>BTPMakeBillFromUI_toolbar</name>
    <classType>com.kingdee.bos.ctrl.swing.KDToolbar</classType>
    <isContainer>true</isContainer>
    <constraints>10,10,300,150</constraints>
    <tabIndex>0</tabIndex>
    <herited>false</herited>
    <layoutInfo>
        <name>kDLayout1</name>
        <type>KDLayout</type>
        <value />
    </layoutInfo>
</component>
<component>
    <name>BTPMakeBillFromUI_menubar</name>
    <classType>com.kingdee.bos.ctrl.swing.KDMenubar</classType>
    <isContainer>true</isContainer>
    <constraints>10,10,300,150</constraints>
    <tabIndex>0</tabIndex>
    <herited>false</herited>
    <layoutInfo>
        <name>kDLayout1</name>
        <type>KDLayout</type>
        <value />
    </layoutInfo>
</component>

D）、在所有<component>节点的孩子节点中，找到如下节点：
<parent>
    <key name="name" value="BTPMakeBillFromUI" />
</parent>
将其中的BTPMakeBillFromUI替换成CoreUI

E）、在所有节点的孩子节点中，找到如下节点：
<parent>
    <key name="name" value="BTPMakeBillFromUI_toolbar" />
</parent>
将其中的BTPMakeBillFromUI_toolbar替换成CoreUI_toolbar

F）、在所有<component>节点中，找到如下节点：
<parent>
    <key name="name" value="BTPMakeBillFromUI_menubar" />
</parent>
将其中的BTPMakeBillFromUI_menubar替换成CoreUI_menubar

G）、最后，将修改过的文件以UTF-8文件格式保存即可。
