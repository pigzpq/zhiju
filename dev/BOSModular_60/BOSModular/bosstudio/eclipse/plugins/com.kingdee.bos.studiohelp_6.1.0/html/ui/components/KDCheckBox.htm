使用指南

       复选框控件扩展了状态属性：

Selected属性：按钮的选择状态，使用int型变量：

    ・UNSELECTED 未选

    ・SELECTED 全选

    ・PARTSELECTED 部分选

    同时JCheckBox的Selected属性仍然可以使用。

Switch属性：按钮的开关风格和三态风格属性。开关风格下按钮只有两个状态；三态风格多了部分选的状态。

AlterType状态转换方式属性：有开关转换方式和三态转换方式两种：

    ・ALTER_TWOSTATE 开关转换方式

    ・ALTER_FULLSTATE 三态转换方式
