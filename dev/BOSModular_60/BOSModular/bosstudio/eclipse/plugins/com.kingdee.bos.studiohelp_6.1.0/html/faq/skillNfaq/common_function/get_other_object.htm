如何从一个编辑器(Editor)中快速定位其它的对象

如何从一个编辑器(Editor)中快速定位其它的对象？比如，如何从实体中看到对应的表对象？

我们在一个Editor中，经常需要查看与该Editor相关的其它Editor。譬如在实体MyEntity定义界面中，想快速的定位到MyEntity对应的表对象T_MyTable，可以如下操作：在对应表对象的TextEdit框中，鼠标悬停，会发现有提示：hold CTRL and click，就是说：如果用鼠标点一下TextEdit框（先让该TextEdit框得到焦点），然后按住Ctrl键，TextEdit框中的字会变成超链形式（字体变蓝并加下划线），然后点击鼠标，即可打开该TextEdit框所描述的文件。
