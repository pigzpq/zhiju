★     关于KDList及其模

1、  KDList继承于JList。

2、  使用KingdeeListModel（它继承于DefaultListModel，增加了图标等数据）。

3、  实现DefaultKingdeeListCellRenderer绘制带图标的选项。

 

使用KDList，没有必要设置模，只须new创建并参考下面的“增删选项”。

例：

String[] s = {"Item1"," Item2"," Item3"};

KDList list = new KDList(s);

例：

KDList list = new KDList();

 

★     增删选项（每个选项是一个Object）

    /** 在列表末端增加一个选项 */

public void addElement(Object elem)

/** 在列表中指定位置插入一个选项 */

public void insertElement(Object elem, int index)

/** 在列表中指定位置重置选项，返回原选项 */

public Object setElement(Object elem, int index)

 

    /** 在列表中移除选项 */

public boolean removeElement(Object elem)

/** 在列表中移除指定位置的选项 */

public void removeElementAt(int index)

/** 移除列表中所有选项 */

public void removeAllElements()

 

/** 取得列表中指定位置的选项 */

public Object getElement(int index)

/** 取得选项在列表中的位置 */

public int getIndexOfElement(Object elem)

/** 判断选项是否在列表中 */

public boolean isContainsElement(Object elem)

/** 获取选项个数 */

public int getElementCount()

 

例：

list.addElement(“Item4”);        //列表末尾追加选项

list.insertElement(“Item0”, 0);  //插入选项

    public void setElementIcon(Icon icon, Object elem)

    /** 取得索引为index的选项的图标 */

例：

list.setElementIcon(icon0, 0);       //可以是指定index设置图标

list.setElementIcon(icon1, “Item1”); //也可以指定Object(选项本身)

 

★     选项多列布局

通过设置选项的布局方向和显示行数来实现：

public void setLayoutOrientation(int layoutOrientation)

public void setVisibleRowCount(int visibleRowCount)

其中，layoutOrientation 取如下值：

JList.VERTICAL（选项排布如下）

        0

        1

        2

        3

JList.HORIZONTAL_WRAP（选项排布如下）

        0  1

        2  3

JList.VERTICAL_WRAP（选项排布如下）

        0  2

        1  3

 

例：

list.setVisibleRowCount(5);

list.setLayoutOrientation(KDList.HORIZONTAL_WRAP);

★     事件（参考JDK帮助文档）

监听选中项改变

例：

        list.addListSelectionListener(new ListSelectionListener()

        {

            public void valueChanged(ListSelectionEvent e)

            {

                if(!e.getValueIsAdjusting())

                {

                    System.out.println(((JList)e.getSource()).getSelectedValue());

                }

            }

});

 

监听鼠标事件（通过鼠标位置得到选项索引）

例：

        list.addMouseListener(new MouseAdapter()

        {

            public void mousePressed(MouseEvent e)

            {

                Point p = new Point(e.getX(), e.getY());

                System.out.println(list.locationToIndex(p));

            }

        });
