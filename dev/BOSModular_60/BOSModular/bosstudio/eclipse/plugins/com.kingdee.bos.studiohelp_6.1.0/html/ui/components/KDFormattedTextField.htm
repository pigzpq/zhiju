功能列表

1、  只能输入合法的数字

2、  提供整数或浮点的输入（可以设置KDFormatTextedField的类型）

3、  浮点型提供精度格式化（以及负数文本颜色的变化）

使用指南

KDFormatTextField是一个从JFormatTextField继承的控件，但是JFormatTextField是一个功能比较多的控件,KDFormatTextFiled利用其功能来实现我们所要的效果

 

KDFormatTextFiled提供两种类型（INTEGER、DECIMAL），在构造函数的时候指定其类型

整型的KDFormatTextFiled提供了输入校验的功能；浮点型的还提供精度设置（自动格式化）及附数颜色控制

 

例子1： 整数型输入框

KDFormatTextField integerField=

new KDFormatTextField(KDFormatTextField.INTEGER);

 

例子2： 浮点型输入框

    KDFormatTextField floatField=

new KDFormatTextField(KDFormatTextField.DECIMAL);

floatField.setPrecision(5);//精确到小数点后第5位

floatField.setNegativeColor(Color.red);//设置负数的颜色
