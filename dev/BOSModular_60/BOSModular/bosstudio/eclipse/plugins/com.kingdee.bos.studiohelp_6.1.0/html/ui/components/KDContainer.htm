Tab键切换内容面板中的元素

使用指南

       可展容器是一个特殊的容器，特点是可伸缩。下面是该控件的使用方法。

       属性的使用：

・State属性：容器的状态属性，有展开和收缩两种状态：

        1）SHRINK 

    2）SPREAD

・EnableActive：设置容器是否可以伸缩活动。值为false时不能伸展或收缩。

・Title属性：标题属性，为文本。

・TitleLength：标题长度。如果不满意默认的标题长度，可以自定义长度。默认值为TITLELABEL_MAXLEN。

・TitleStyle：设置标题的风格。有两种：可展按钮风格（ACTIVESTYLE）、主界面风格（STATICSTYLE）、标签风格（LABELSTYLE）。如下图：

主界面风格：

有一个几何面板作为装饰

可展按钮风格：

按钮和标签组成，按钮是箭头按钮

标签风格:

纯文本标签，比主界面风格少了几何装饰面板，同样当EnableActive为false时，图标变为箭头表示可展。

 

・Gap属性：设置最右边的命令按钮（标题条右侧）与容器右边框的距离。标签风格风格默认距离为0像素，其他风格默认为8像素。setGap(-1)表示使用默认距离。

 

方法的使用：

・add(Action)：在标题条上添加一个按钮，用法同在JToolBar上添加一个按钮。

・removeButton(JButton): 移除标题条上的一个按钮，参数是add(Action)方法中返回的JButton对象。

・removeAllButton: 移除标题条上所有的按钮。

 

使用KDContainer时建议使用KDContainer.add(Action)方法添加标题条按钮，使用方法和ToolBar的add(Action)一样。示范如下：

 Action deleteAction = new DeleteAction("Delete", new TestIcon("D"));
 kdContainer.add(deleteAction);

...

而addButton是为添加一些自定义的按钮准备的接口。

 

在可展容器中放置控件：

为了在容器上放置控件，需要使用getContentPane()获得内容面板，添加控件和设置布局管理器。

 

事件响应：

可展容器发出活动容器事件：

public void addActiveContainerListener(ActiveContainerListener);

public void removeActiveContainerListener(ActiveContainerListener);

其中ActiveContainerListener用于处理容器事件，包括以下事件类型：

・SPREAD展开命令

・SHRINK 收缩命令

在容器形状发生变化时，客户程序需要刷新布局；当最大化事件发生时，客户程序需要根据当前父容器的大小和排布方式重新设置该容器控件的大小和其它子控件的大小，并最后刷新父容器布局。
