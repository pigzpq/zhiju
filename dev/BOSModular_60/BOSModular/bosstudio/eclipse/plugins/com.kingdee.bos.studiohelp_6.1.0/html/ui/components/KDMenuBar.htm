功能列表

 请注意当你看到这句话时，菜单栏还没实现向IE浏览器那样的功能，在6月30号之前完成。

 

菜单栏（KDMenuBar）的布局方式IE 浏览器一样。

 

菜单栏(com.kingdee.bos.ctrl.swing.KDMenuBar) 添加的菜单默认情况下是靠左布局的，但现在提供扩展接口<code>addRightComponent(JComponent menu,int rightInterval)</code> 可将一个Swing控件插入到靠右端的位置。应用场景如:在菜单栏的右边插入一个帮助按钮（参见EAS3.0的主界面）(在4月30号之前提供)

菜单栏可以添加一个Logo , 参见接口<code>setLogo(Icon)</code> (在4月30号之前提供)

菜单栏可在同一个方位区上移动，移动的方式为：与工具栏进行交换行显示或者交换列显示。

菜单栏上的菜单(KDMenu)，全部显示或隐藏显示；

当窗口大小改变时：()

1，  菜单栏在北区、南区时，窗口的宽度不足以水平的显示菜单栏上的所有菜单，则菜单栏的高度改变，菜单在菜单栏上隐藏一部分显示； (在4月30号之前提供)

2，  菜单栏在东区、西区时，窗口的高度不足以垂直的显示菜单栏上的所有菜单，则菜单栏的宽度改变，菜单在菜单栏上隐藏一部分菜单显示；  (在4月30号之前提供)

菜单栏以浮动窗口方式显示 。(在4月30号之前提供)

菜单栏可拖拉到四个区进行布局

菜单的快捷键操作与word和永中office一样

1：Alt键 激活菜单栏上的第一个菜单（比如word中的“文件(F)”）

2：在激活第一个菜单后，可用“左右箭头”在菜单栏上导航菜单。

3：在激活第一个菜单后，可由“相关键“直接选中某个菜单，并弹出该菜单。比如按下“E”，会把“编辑(E)”菜单击中，并弹出“编辑(E)”菜单。

4：在选中一个菜单后，可用“上下箭头”进行导航菜单项（MenuItem）

 

5：Alt＋“某键”，选中某菜单，并弹出某菜单。比如同时按下“Alt+F”键，击中“文件（F）”菜单，并弹出该菜单。

默认情况下菜单是从左至右布局，提供接口把指定的菜单从右至左布局。

 

菜单的行为方式，菜单项的行为方式，与标准JMenu，JMenuItem完全兼容。比如

1：菜单的打开。

2：根据index从MenuBar上得到指定的Menu。

3，根据index从Jmenu上得到JmenuItem。等等。

 

KDMenuBar从javax.swing.JMenuBar继承。

KingdeeMenuBarUI从BasicMenuBarUI继承。

仅构造新的布局管理器，来完成KDMenuBar的要求。

 

使用指南

 

//1 构造菜单栏

    private KDMenuBar getKDMenuBar() {

       if (menuBar == null) {

           menuBar = new KDMenuBar();

 

           //1 File

           KDMenu menuFile = new KDMenu("File");

           KDMenuItem itemNew = new KDMenuItem("New");

           KDMenuItem itemSave = new KDMenuItem("Save");

           KDMenuItem itemSaveAs = new KDMenuItem("saveAs");

 

           menuFile.add(itemNew);

           menuFile.add(itemSave);

           menuFile.add(itemSaveAs);

 

           //2

           KDMenu menuEdit = new KDMenu("Edit");

           KDMenuItem itemUndo = new KDMenuItem("Undo");

           KDMenuItem itemRedo = new KDMenuItem("Redo");

           KDMenuItem itemCut = new KDMenuItem("Cut");

           menuEdit.add(itemUndo);

           menuEdit.add(itemRedo);

           menuEdit.add(itemCut);

 

 

           //6

           menuBar.add(menuFile);

           menuBar.add(menuEdit);

    。。。。

 

       }

       return menuBar;

   }

 

// 给窗体设置菜单栏

KDFrame mainframe = new KDFrame();

mainFrame.setJMenuBar(getKDMenuBar());
