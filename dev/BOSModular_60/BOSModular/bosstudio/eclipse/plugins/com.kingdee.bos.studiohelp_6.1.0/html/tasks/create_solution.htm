创建一个解决方案

针对BOS而言，解决方案是元数据的最大载体，所有的元数据都必须位于一个解决方案之内，譬如，EAS就是一个解决方案；从Eclipse的角度来说，解决方案就是Eclipse的一个Project。因此，解决方案本身在BOS Studio中就具备Project的一些性质，譬如导入、导出、刷新等。

在BOS Studio中，解决方案能够新建、打开、关闭、删除。这些操作均通过“BOS资源导航视图”完成。

如何创建解决方案

1、打开“包分类”视图，在右键菜单中选择“新建”->“解决方案”，弹出“解决方案创建向导”，如图一：

图一

2、输入解决方案的“名称”、“别名”和“描述”，确定解决方案的放置位置，点击“下一步”，如图二：

图二

3、在“目录(D)”栏中输入元数据放置的目录，点击“完成”，即可建立一个解决方案。在“包分类”视图中可看见结果，如图三：
